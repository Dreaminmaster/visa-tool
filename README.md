# 🌍 多国签证申请表填写工具

> 为中国公民量身定制的签证申请表在线填写工具

## ✨ 功能特点

- **首页国家选择**：直观的卡片式国家选择器，支持区域筛选
- **多国支持**：覆盖申根区、亚洲、美洲、大洋洲主要目的地
- **PDF导出**：支持导出填写完成的官方PDF申请表
- **数据保存**：自动保存填写内容，刷新不丢失
- **中文界面**：全中文操作界面，字段中英双语标注

## 🚀 使用方法

1. 打开网站：https://dreaminmaster.github.io/visa-tool/
2. 在首页选择目的地国家
3. 填写个人信息（中英双语）
4. 填写旅行信息
5. 确认并导出PDF

## 📋 表单内容

- **个人基本信息**：姓名、出生日期、国籍、性别、婚姻状况
- **旅行证件**：护照号码、签发日期、有效期
- **联系方式**：地址、电话、电子邮件
- **职业信息**：工作单位、职位
- **旅行详情**：目的地、入境日期、停留时间
- **费用承担**：自付或赞助方

## 🛠️ 技术栈

- HTML/CSS/JavaScript（单文件应用）
- [pdf-lib](https://pdf-lib.js.org/)（PDF表单填写）
- 无需后端，纯前端实现

## 📝 注意事项

- 本工具仅用于辅助填写申请表，不保证签证获批
- 请务必核对所有信息的准确性
- 部分国家需要通过官方系统在线申请
- 建议使用最新版Chrome/Firefox/Safari浏览器
